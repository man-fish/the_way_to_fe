### 1.目录结构

```cmd
C:
└─miniprogram-test-1
    │  
    ├─assets	//静态文件（最后要部署到服务器上）
    │  ├─icons
    │  │      grid-01.png
    │  │      
    │  ├─images
    │  │      avatar.png
    │  │      
    │  └─tabs
    │          home-active.png
    │          home.png
    │          
    ├─pages
    │  ├─index
    │  │      index.js
    │  │      index.json
    │  │      index.wxml
    │  │      index.wxss
    │  │      
    │  ├─logs
    │  │      logs.js
    │  │      logs.json
    │  │      logs.wxml
    │  │      logs.wxss
    │  │      
    │  └─message
    │          message.js
    │          message.json
    │          message.wxml
    │          message.wxss
    │         
    │  app.js	//暂时不知为何物
    │  app.json		//全局配置，详情见下方
    │  app.wxss			//全局样式，就是说写在这里的样式全局都有效
    │  project.config.json		//更新或者上传的时候用的配置文件，无大用。
    |
    └─utils
        └─util.js	//暂时不知为何物
```
