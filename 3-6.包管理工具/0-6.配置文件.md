## 删除

```bash
> npm config rm <configPropName>
```

## 代理

更改 http 代理：

```bash
> npm config set proxy=http://127.0.0.1:8087
```

配置 https 代理：

```bash
> npm config set https-proxy https://server:port
```

## 注册

配置注册表`即依赖包数据库地址`：

```bash
npm config set registry https://registry.npm.taobao.org/
```
